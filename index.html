等额本金

月供组成：本金 + 未还本金的利息；

每月应还本金算法：贷款总额 / 贷款期数；
未还本金利息算法：详见案例分析

案例分析： 陈某向银行贷了 120 万元，年限为 10年[120个月]，年利率为央行基准利率[4.9%]；

月利率： 0.049 / 12 = 0.00408333

本金： 那么他每月还的本金就应为： 120w / 120月 = 1w/月 利息： 第一个月就应是：120w * 0.00408333 = 4899.96； 第二个月就应是：( 120w - 1w ) * 0.00408333 = 4859.17 第三个月就应是：( 120w - 2w ) * 0.00408333 = 4818.33 ……………………………………………………………………… 第120个月就应是： ( 120w - 119w ) * 0.00408333 = 40.84

总结：每月还的本金是固定的，并不会变化；但是每月的利息是根据最新剩余本金来计算，因此，利息每个月呈递减的趋势发展，所以整体月供 也呈递减趋势。

等额本息

案例同上

根据银行利率对应月供表来计算 月供：每月固定不变 但是，利息, 逐月递减；本金，逐月递增

如果是信贷的话：就根据平均利息来计算。
例如：王某在银行办理了一笔 额度为 10万元的信用贷款，年限为3年，平均利息为8.5厘。 每月应还本金： 10w / 36月 = 2777元/月 每月应还利息： 10w * 0.0085 = 850元/月 因此，月供为： 2777元 + 850元 = 3627元/月
